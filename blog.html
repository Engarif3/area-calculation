Questions & Answers

  1. 1) What is DOM and what is the purpose of DOM?

    Dom stands for Document Object Model. It represents HTML and XML documents as a structure of nodes and objects.

    In other way we can say that DOM represents the logical structure of documents and how they are accessed and manipulated. DOM helps us to create, update, and delete the document structure, style, and content.

  3. 3) What is event bubble?

    Event Bubbling is a term used in DOM (Document Object Model). It occurs when an element is received as an event, and that event goes up or bubbles up (or can be said that how it is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it finds the root element.



  4. 4) What is a callback function and why will you use it?

    A callback is a function that is passed as an argument to another function.It should be called when a certain condition is met. It does not get called immediately rather the callback function is called after a certain period of time.
